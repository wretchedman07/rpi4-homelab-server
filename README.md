# Raspberry Pi 4 Homelab Server

## Overview

This repository documents my Raspberry Pi 4 homelab built to learn Linux system administration, Docker, networking, and self-hosting.

## Hardware

- Raspberry Pi 4
- Raspberry Pi OS
- External Seagate 1TB Drive
- SSH using Termius
- Tailscale VPN

## Services

- Jellyfin
- Portainer
- Heimdall
- Uptime Kuma
- n8n
- Samba

## Storage

- External NTFS drive
- Mounted permanently using /etc/fstab
- Mount point: /mnt/media

## Goals

- Learn Linux
- Learn Docker
- Build implementation skills
- Build Solutions Architect skills
- Document everything
